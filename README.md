# MakeupGame — Тестовое задание Unity

Игровая механика нанесения макияжа (крем, тени, помада, румяна) для мобильных устройств (Android).

## Механика

- Игрок **нажимает на инструмент** → рука автоматически берёт его
- Игрок **тащит руку к лицу** (Drag)
- При попадании в зону лица → **применяется эффект**, рука возвращается на место
- Отпустить вне зоны лица — ничего не происходит
- **Голубой спонжик** сбрасывает весь макияж

### Инструменты

| Инструмент | Механика |
|---|---|
| Крем | Убирает прыщи |
| Тени | Кисточка окунается в выбранный цвет → наносится на веки |
| Помада | Меняет цвет губ |
| Румяна | Кисточка окунается в выбранный цвет → наносится на щёки |

## Технологии

- **Unity** 2022.3 (Android)
- **Zenject** — Dependency Injection, вся архитектура построена через DI-контейнер
- **DOTween** — плавные анимации движения руки между точками

## Архитектура

Проект построен с соблюдением принципов **SOLID**:

- **SRP** — каждый класс отвечает за одну задачу (`Hand` — движение, `FaceZone` — детектирование, `MakeupService` — хранение состояния)
- **OCP** — новый инструмент добавляется без изменения существующих классов, достаточно реализовать `ITool`
- **LSP** — любой `BaseTool` подставляется вместо `ITool` без побочных эффектов
- **ISP** — инструменты с шагом подготовки реализуют отдельный `IPreparable`, остальные — нет
- **DIP** — `Hand` и `FaceZone` зависят от `ITool` (абстракция), не от конкретных инструментов

```
│   ├── LipstickTool.cs

├── Services/
│   ├── IMakeupService.cs
│   └── MakeupService.cs      — хранит выбранный макияж по категориям
├── Controllers/
│   ├── MakeupController.cs   — медиатор между UI и инструментами
│   ├── DragController.cs     — управление drag-сессией
│   └── InputHandler.cs       — mouse / touch input
├── UI/
│   ├── MakeupTabsView.cs     — вкладки категорий
│   ├── ColorPaletteView.cs   — палитра цветов
│   ├── ColorItemView.cs      — кнопка цвета
│   ├── FaceApplier.cs        — отображение результата на лице
│   └── SpongeButton.cs       — сброс макияжа
└── Installers/
    └── GameInstaller.cs      — Zenject биндинги
```
